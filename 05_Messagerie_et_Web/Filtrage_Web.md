# Filtrage Web

## Objectif

Controler les acces Internet afin de proteger les utilisateurs, limiter les telechargements dangereux et reduire l'exposition du systeme d'information.

## Categories Bloquees

- malware et phishing ;
- telechargement d'executables ;
- piratage et cracks logiciels ;
- anonymiseurs et proxys ;
- VPN personnels ;
- hebergement de fichiers non approuve ;
- messageries personnelles lorsqu'elles servent a traiter des donnees hospitalieres ;
- sites identifies comme a risque par les outils de reputation.

## Telechargements

Les telechargements de logiciels sont bloques par defaut. Toute installation doit passer par la DSI et le catalogue logiciel approuve.

Les fichiers telecharges autorises restent soumis a analyse antivirus ou EDR.

## Wi-Fi Patients

Le Wi-Fi patients est filtre et isole. Il donne uniquement acces a Internet et ne permet aucun acces aux ressources internes :

- pas d'acces au DPI ;
- pas d'acces aux serveurs ;
- pas d'acces aux imprimantes ;
- pas d'acces aux postes administratifs ou medicaux.

## Derogations

Une derogation de filtrage doit etre limitee dans le temps, justifiee par un besoin professionnel et validee par la DSI. Les derogations sont revues mensuellement.

## Indicateurs

- nombre de blocages malveillants ;
- nombre de telechargements bloques ;
- nombre de derogations actives ;
- nombre d'alertes utilisateur ;
- categories les plus sollicitees.
