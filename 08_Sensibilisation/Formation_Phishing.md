# Formation Phishing

## Objectif

Apprendre aux agents a detecter, signaler et eviter les courriels frauduleux pouvant conduire a un vol d'identifiants, une fuite de donnees ou un ransomware.

## Public Prioritaire

- admissions ;
- facturation ;
- secretariats medicaux ;
- RH ;
- direction ;
- services recevant de nombreuses pieces jointes externes.

## Messages Cles

- un expediteur connu peut etre usurpe ;
- une urgence artificielle est un signal d'alerte ;
- une piece jointe inattendue doit etre verifiee ;
- un lien demandant un mot de passe est suspect ;
- le signalement rapide protege l'ensemble de l'hopital.

## Reflexes

1. verifier l'expediteur et le domaine ;
2. se demander si la demande est attendue ;
3. ne pas ouvrir une piece jointe douteuse ;
4. ne jamais saisir son mot de passe depuis un lien recu par mail ;
5. signaler le message a la DSI.

## Exercices

Des exemples fictifs sont utilises : fausse facture fournisseur, fausse convocation RH, fausse notification Microsoft 365, faux resultat de laboratoire.

## Evaluation

La formation se termine par un quiz court et un exercice de signalement.
