# Gestion des Cles USB

## Objectif

Prevenir l'introduction de logiciels malveillants et la fuite de donnees via supports amovibles.

## Regle Generale

Les ports USB de stockage sont bloques par defaut sur les postes du Centre Hospitalier des Calanques.

## Interdictions

- utiliser une cle USB personnelle ;
- connecter un support trouve ou inconnu ;
- copier des donnees patients sur un support non chiffre ;
- contourner le blocage USB ;
- utiliser une cle USB pour transferer des logiciels non valides.

## Derogation

Une derogation peut etre accordee pour un besoin metier documente, par exemple un equipement biomedical ou un transfert ponctuel sans alternative.

La demande doit preciser :

- service demandeur ;
- justification ;
- type de donnees ;
- duree ;
- poste concerne ;
- responsable metier ;
- validation DSI.

## Mesures de Securite

- support fourni ou valide par la DSI ;
- chiffrement si donnees sensibles ;
- analyse antivirus avant utilisation ;
- journalisation de l'usage ;
- date de fin de derogation.

## Revue

Les derogations actives sont revues mensuellement. Toute derogation expiree est retiree.
